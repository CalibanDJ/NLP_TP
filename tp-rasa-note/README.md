## Usage

The goal of this chat bot is to inform the user about Price/Range/Change of a crypto that is in the Top 100 world


In order to use it, run :

```sh
rasa train
rasa run actions
rasa shell
```

## Commannds

The bot wait for a crypto like "Bitcoin" and will ask for which details the user want to know.
